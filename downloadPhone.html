The phone version doesn't have a UI, you have to place the block on the last place in your hotbar like this:
(NO BLOCK = AIR)
The first block is a ice block, use this to set positions.
The second block is a packed ice block use this to fill the area.
The third block is a blue ice block use this to set the clone position.
If the clone position is set it will automatically clone it.

You can get the blocks from the inventory or from the chat command /function wand
